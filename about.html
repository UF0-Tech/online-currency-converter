About This Currency Converter

This Currency Converter, developed by Chuka Uwefoh, enables users to convert between global currencies using live exchange rates updated daily. The project began as a desktop application and was later evolved into a web application to enhance accessibility and usability. Future plans include expanding the converter into a mobile application with additional features to further improve the user experience.

It was designed to be simple, intuitive, and responsive so you can easily use it on both desktop and mobile devices.

Built with HTML, CSS, and JavaScript with exchange rates fetched from an exchange rate API.
